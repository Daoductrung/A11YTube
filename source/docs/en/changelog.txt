Version 0.1
Release Status: Initial Release
Welcome to the first version of A11YTube, an accessible application designed to provide a streamlined, keyboard-friendly experience for browsing, playing, and downloading content from YouTube.
New Features:
- User Interface:
  + Tabbed interface (Home, Tools, Help) accessible via Control+Tab.
  + Multilingual support: Available in English and Vietnamese.
- Search & Navigation:
  + Search for YouTube videos and playlists (Ctrl+F) with filter options.
  + "Load More" functionality: Automatically load more results when reaching the end of the list.
- Media Player:
  + Accessible player with shortcuts for Play/Pause (Space), Seek, and Volume.
  + Speed controls (Slow/Normal/Fast).
  + Playback preferences: Options to toggle "Continue watching", "Repeat track", and "Auto play next track".
  + Support for video descriptions (Ctrl+Shift+D).
- Download Manager:
  + Direct Download (Ctrl+D) and Context Menu downloads.
  + Format support: Video (MP4) and Audio (M4A, MP3).
  + MP3 Conversion Quality options: Selectable bitrates (96 kbps, 128 kbps, 192 kbps).
  + Custom download folder selection via Settings.
- Library & Tools:
  + Favorites system (Ctrl+Shift+F) and Watch History (Ctrl+H).
  + Auto-detect links from clipboard on startup.
  + Auto-check for updates on startup.
- Global Shortcuts:
  + Comprehensive hotkey support (e.g., Ctrl+Y to Play from Link, Ctrl+P to Open Downloads Folder).
